# נוירואנטומיה: מנחים, חתכים ומבנים מרכזיים

## CNS ו-PNS

| מערכת | שם מלא | מרכיבים |
|-------|--------|---------|
| **CNS** | Central Nervous System | מוח וחוט השדרה |
| **PNS** | Peripheral Nervous System | כל העצבים שמחוץ ל-CNS |

---

## למה מנחים וחתכים חשובים?

תמונות מוח (MRI, חתכים, איורים) תמיד מתוארות ביחס למנחים וחתכים.

!!! warning "שים לב"
    טעות בזיהוי כיוון או חתך → טעות בפרשנות השאלה במבחן!

---

## מנחים אנטומיים (Directional Terms)

### מנחים בסיסיים

| מונח | מונח נוסף | משמעות |
|------|-----------|--------|
| **Anterior** | Rostral | קדמי, לכיוון האף |
| **Posterior** | Caudal | אחורי, לכיוון העורף |
| **Dorsal** | - | לכיוון הגב; במוח האנושי ≈ למעלה |
| **Ventral** | - | לכיוון הבטן; במוח האנושי ≈ למטה |
| **Medial** | - | קרוב לקו האמצע (midline) |
| **Lateral** | - | רחוק מקו האמצע (לכיוון הצדדים) |
| **Superior** | - | עליון |
| **Inferior** | - | תחתון |

### תרשים מנחים

```
                Superior (עליון)
                     ↑
                     │
    Lateral ←────────┼────────→ Medial
   (צידי)            │           (מרכזי)
                     │
                     ↓
                Inferior (תחתון)

        Anterior ←───────→ Posterior
        (קדמי)              (אחורי)
```

## מבטים על המוח (Views)

| מבט | שם באנגלית | מה רואים |
|-----|------------|----------|
| מבט מהצד | **Lateral view** | המיספרה אחת |
| מבט פנימי | **Medial view** | לאחר חצייה בין ההמיספרות |
| מבט מלמעלה | **Superior view** | שתי ההמיספרות יחד |
| מבט מלמטה | **Inferior view** | בסיס המוח |

## חתכים במוח (Planes / Sections)

![חתכי מוח](../assets/brain_slices.png)

### סוגי חתכים

| חתך | שם נוסף | כיוון החיתוך | מה רואים |
|-----|---------|-------------|----------|
| **Sagittal** | - | ימין-שמאל | המיספרה אחת מהצד |
| **Mid-sagittal** | - | על קו האמצע | מבנים מדיאליים (תלמוס, גזע מוח) |
| **Coronal** | Frontal | קדמי-אחורי | שתי המיספרות ("פנים") |
| **Horizontal** | Axial | עליון-תחתון | "פרוסות" מלמעלה למטה |

!!! tip "טיפ למבחן"
    אם רואים מבנים מדיאליים (כמו תלמוס או גזע מוח) – זה כמעט תמיד **mid-sagittal**.

## המיספרות וקישוריות

| מונח | שם באנגלית | משמעות |
|------|------------|--------|
| **נגדי** | Contralateral | קלט/פלט מהצד הנגדי |
| **אותו צד** | Ipsilateral | קלט/פלט מאותו צד |

!!! info "עיקרון חשוב"
    כל המיספרה שולטת בעיקר על **הצד הנגדי** של הגוף.

## מבנים מרכזיים

### תלמוס (Thalamus)

- מבנה **עמוק ומרכזי** במוח
- משמש כ**תחנת ממסר עיקרית** למידע סנסורי לקורטקס
- כמעט כל המידע החושי עובר דרכו (חוץ מריח)

### צרבלום (Cerebellum)

- נמצא **מאחור ולמטה** ("מוח קטן")
- בעל קפלים צפופים, נראה כמו "אגוז"
- אחראי על:
    - תיאום תנועה
    - דיוק ותזמון
    - **לא יוזם תנועה** – רק משפר אותה

!!! warning "שאלה נפוצה במבחן"
    הצרבלום ≠ גזע המוח. הם מבנים נפרדים!

---

## גזע המוח (Brainstem)

סדר אנטומי מלמטה למעלה:

```
Spinal cord → Medulla → Pons → Midbrain
   (חוט שדרה)   (מדולה)   (פונס)  (מוח תיכון)
```

| מבנה | תפקידים עיקריים |
|------|-----------------|
| **Medulla** | נשימה, קצב לב, לחץ דם |
| **Pons** | ויסות נשימה, שינה, חיבור לצרבלום |
| **Midbrain** | רפלקסים חזותיים/שמיעתיים, בקרת תנועה בסיסית |

!!! danger "קריטי למבחן"
    פגיעה ב-**Medulla** עלולה להיות **קטלנית** – שולט בתפקודים חיוניים!

## קליפת המוח: Gyri ו-Sulci

| מונח | יחיד | משמעות |
|------|------|--------|
| **Gyri** | Gyrus | קפלים |
| **Sulci** | Sulcus | חריצים |

במהלך האבולוציה: יותר קפלים → יותר שטח קורטיקלי → יכולות קוגניטיביות מתקדמות יותר.

---

## חדרי המוח ו-CSF

### חדרי המוח (Ventricles)

- 4 חדרים: שני לטרליים, שלישי, רביעי
- **לא רואים חדרים במבט חיצוני** – הם פנימיים

### CSF (Cerebrospinal Fluid)

- נוזל שנוצר בחדרים
- תפקיד: הגנה, בלימת זעזועים
- נמצא ב-**Subarachnoid space**

---

## קרומי המוח (Meninges)

מהחוץ פנימה:

| קרום | שם באנגלית | מאפיינים |
|------|------------|----------|
| 1 | **Dura mater** | קשה וחיצונית |
| 2 | **Arachnoid** | קרום אמצעי |
| 3 | **Pia mater** | דק ונצמד למוח |

---

## טעויות נפוצות במבחן

| טעות | תיקון |
|------|-------|
| בלבול בין medial ל-lateral | medial = מרכז, lateral = צדדים |
| בלבול בין dorsal/ventral לבין superior/inferior | במוח הם דומים, אבל לא זהים! |
| זיהוי חתך coronal כ-sagittal | ב-coronal רואים **שתי המיספרות** יחד |
